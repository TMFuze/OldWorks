разработать групповые запросы
1. Вычислить кол-во значений в некоторой таблице.
2. Вычислить 2 или более статистических функции для таблицы.
3. Вычислить статистическую информацию относительно с группированных данных по некоторому полю.
4. Используя 2 и более статистических функции вычислить даннные по некоторому сгрупированному полю.
5. Вычислить статистическую информацию по двум и более критериям группировки.
6. Вычислить статистическую информацию по группам и ограничить её по некоторому критерию отбора.
7. Вычислить статистическую информацию по группам при этом сами данные ограничены критерием отбора.
8. Проанализировать вывод статистической информации при отсутствии данных.
